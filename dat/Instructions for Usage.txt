1.	Update the Incfiles\original folder with the latest inc files.
2.	Execute PreProcess.bat. This will update the inc files in the incfiles folder.
3.	Update CriticalChanges.txt
4.	Update the CSV file with the correct chip data
5.	Execute getchipdata.exe > chipfiles/outputlog.txt
6.	Copy to chipfiles to distribution
7.	Update SVN:DAT


ConfigBaseLoc is basically the last location for the config, minus 15
On the 18F67J50, the last location is 0x1FFFF, so the ConfigBaseLoc is 0x1FFFF - 15 = 0x1FFF0
Normal location for an 18F is from 0x300000.
